# Завдання 1

1. Реалізувати структуру даних «Стек», застосувавши всі зазначені методи. Написати програму (функцію main), що містить опис стека, операції та виконує поставлене завдання. Розмір стека (S) визначити за формулою S = варіант_здобувача * 5 + 50; 
2. Заповнити стек випадково згенерували числами від 1 до 1000. 
3. Створити 2 нових стека й перенести всі парні числа з початкового стека в 1-й стек, усі непарні – у 2-й. 
4. Вивести в консоль елементи «парного» й «непарного» стеків.
