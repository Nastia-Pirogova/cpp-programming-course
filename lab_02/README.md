# Завдання 1

1. Реалізувати структуру даних «Черга», застосувавши всі методи. Написати програму (функцію main), що містить опис структури та моделює роботу черги. Розмір черги (S) визначити в такий спосіб:
S = варіант_здобувача * 5 + 50; 
2. Заповнити чергу випадково згенерували числами від 1 до 1000. 
3. Вивести в консоль елементи черги, що є простими числами.
