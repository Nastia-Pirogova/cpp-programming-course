# Завдання 1

1. Реалізувати структуру даних «Ґеш-таблиця», виконавши  на ній операції вставки, видалення та пошуку елементів. Написати програму (функцію main), що містить опис ґеш-таблиці
із заданими операціями. Розмір ґеш-таблиці (S) визначити як 75 % від кількості елементів (N), що будуть зберігатись та яку визначаємо за формулою N= варіант_здобувача * 5 + 50;
2. Заповнити ґеш-таблицю розміру S випадковими числами, кількість яких N.
3. Видалити з ґеш-таблиці всі парні числа.
4. Вивести в консоль елементи що залишилися.
